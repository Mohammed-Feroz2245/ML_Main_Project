# Online Course Completion Prediction

End-to-end Machine Learning project using:
- Scikit-learn
- FastAPI
- Docker
- AWS ECR & ECS

## Project Overview
Predicts whether a student will complete an online course based on activity and device usage.

## Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- FastAPI
- Docker
- AWS (ECR, ECS)
